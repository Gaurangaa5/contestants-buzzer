# Contestants Buzzer

This is a **Family Game Show buzzer** for use on desktop, tablets, or mobile devices.  

## How to Use

1. Open the buzzer page in a browser:  
   [https://gaurangaa5.github.io/contestants-buzzer/](https://gaurangaa5.github.io/contestants-buzzer/)

2. Press the **Air Horn button** to play the buzzer sound.  

3. The glowing button and sound are designed to work on **all devices**.

## Files in This Repository

- `index.html` – The main buzzer page  
- `buzzer.mp3` – Sound file for the buzzer  
- `LTA-Identity.ttf` – Font used for the button and title  

## Notes

- All files must stay together in the repository for the buzzer to work.  
- Works on desktop and mobile browsers.  
- Make sure to open the **HTML page**, not just the MP3 file.
